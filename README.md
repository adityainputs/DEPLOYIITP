# Dr. Nutan Kumar Tomar - Faculty Website

A clean, fast, and mobile-friendly portfolio website built for **Dr. Nutan Kumar Tomar**, Associate Professor of Mathematics at the Indian Institute of Technology (IIT) Patna.

## 🚀 See it Live
* **Live Website:** https://adityainputs.github.io/DEPLOYIITP/
* **Code Repository:** https://github.com/adityainputs/DEPLOYIITP
* **Built With:** HTML, CSS, and plain JavaScript

---

## ✨ Key Features

* **Professional Design:** Uses IIT Patna's official deep navy and gold colors for a clean, academic look.
* **Mobile-Friendly:** The layout automatically adjusts to look perfect on phones, tablets, and desktop computers. No annoying side-to-side scrolling!
* **Modern Layout:** Replaced old-fashioned, clunky tables with neat, organized "cards" that are much easier to read.
* **Interactive Tabs:** The "Research" and "Thesis" sections use smart tabs. You can click through publications and student lists instantly without waiting for a whole new page to load.
* **Auto-Numbering:** The lists automatically number themselves, making it super easy to add new awards or papers in the future without messing up the counting.

---

## ⚡ Why It's Fast

This site was built with speed and clean code in mind:

1. **One Style File:** All the design rules are kept in a single file (`style1.css`), which helps the website load incredibly fast.
2. **Smart JavaScript:** Instead of loading a bunch of separate pages, the site uses a fast JavaScript engine to swap out information instantly when you click a tab. 
3. **Lightweight:** No heavy outside plugins or libraries were used. It's built entirely from scratch to run as smoothly as possible.
